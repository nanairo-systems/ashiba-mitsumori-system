# モジュールIDインデックス

> 全ページの各セクションにユニークなバッジID（赤ラベル）が付いています。
> 指示の際は **ID だけ** で特定できます（例: 「`SO-3` を修正して」）。

---

## メイン業務システム

### SO — 商談ダイアログ（SiteOpsDialog）
| ID | セクション |
|----|-----------|
| **SO-1** | ヘッダー（現場名・ステータス・会社情報・日程・住所・担当者） |
| **SO-2** | アクションカード（Googleマップ固定・画像登録固定＋カスタム3スロット） |
| **SO-3** | 作業内容タブ（作業グループ切替・追加・編集・削除） |
| **SO-4** | 工程日程（リスト/ガント切替・日程カード・ガントチャート） |
| **SO-5** | 見積詳細（見積一覧・新規見積作成フォーム） |
| **SO-6** | 写真（画像アップロード・管理） |

### CD — 契約詳細（ContractDetail）
| ID | セクション |
